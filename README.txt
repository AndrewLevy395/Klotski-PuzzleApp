Independent Project 2:  Klotski Puzzle Application

Pieces can be selected and moved on board with WASD and arrow keys. When piece is selected it lights up blue. When user moves
win piece (the red piece) through the slot on the bottom of the board, the user wins the game and is rewarded. At any time
the user may choose to restart the puzzle by clicking the restart puzzle button. The number of moves are counted after each
move and are reset when puzzle is reset.

Use Cases:
select piece
move piece
restart puzzle
exit application
hide application
increase window
decrease window
